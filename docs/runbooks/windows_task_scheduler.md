# Windows 任务计划配置

**重要说明**: 本项目不会自动创建任务计划。以下说明帮助用户手动创建。

## 1. 手动创建任务计划

### 1.1 打开任务计划程序
```
Win + R -> taskschd.msc -> 回车
```

### 1.2 创建基本任务
1. 点击"操作" -> "创建基本任务"
2. 名称: `Theme Sector Radar Daily`
3. 描述: `每日盘后板块雷达日报`

### 1.3 设置触发器
1. 选择"每天"
2. 设置时间: `16:30`（建议交易日盘后）
3. 点击"下一步"

**注意**: Windows 任务计划不支持"仅工作日"，需要手动调整或使用高级设置。

### 1.4 设置操作
1. 选择"启动程序"
2. 程序或脚本: `powershell.exe`
3. 添加参数: `-ExecutionPolicy Bypass -File <path-to-a-share-theme-sector-radar>\scripts\run_daily.ps1`
4. 起始于: `<path-to-a-share-theme-sector-radar>`
5. 点击"下一步"

### 1.5 完成
1. 勾选"当点击完成时，打开此任务属性对话框"
2. 点击"完成"
3. 在属性对话框中:
   - 勾选"不管用户是否登录都要运行"
   - 勾选"不存储密码”
   - 取消勾选"只有在计算机使用交流电源时才启动此任务”

## 2. 建议配置

| 配置项 | 建议值 |
|--------|--------|
| 触发时间 | 交易日 16:30 |
| 程序 | powershell.exe |
| 参数 | -ExecutionPolicy Bypass -File scripts\run_daily.ps1 |
| 起始目录 | 项目根目录 |
| 用户账户 | SYSTEM 或当前用户 |

## 3. 测试任务

### 3.1 手动运行测试
1. 在任务计划程序中找到创建的任务
2. 右键 -> "运行"
3. 检查结果

### 3.2 查看历史
1. 右键 -> "属性" -> "历史记录"
2. 查看执行结果

## 4. 注意事项

- **不自动创建**: 本项目不会自动创建任务计划，用户确认后自行创建
- **网络依赖**: 真实 AkShare 模式需要网络连接
- **失败处理**: 任务失败时会记录日志，不会自动重试
- **时区**: 任务计划使用系统时区
